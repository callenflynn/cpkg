# cpkg
my own package manager

## Usage
Using cpkg is simple. Here are some example commands

For a list of commands
``` 
cpkg --help
```
to install simple type
```
cpkg install app.name
```
for example:
```
cpkg install brave-browser.brave
```
to update simply use 
```
cpkg update --all
```
or to update a single app use
```
cpkg update app.name
```
