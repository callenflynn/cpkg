CPKG

Windows packages done differently

What this is

CPKG is a package manager for Windows. It is used to install various apps from your command line.

//just a little bit of convincing :)

What makes us different

We put an emphasis on community and simplicity. Instead of relying on complicated manifests, developers can upload applications by pushing a JSON file to our GitHub repository. If it has a GitHub release, it can be installed.
